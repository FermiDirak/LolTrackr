# LolTrakr

LolTrackr lets competitive League players keep track of their ranked climbing progress, and overall statistics. It also helps keep you informed on all of the latest trends; these include: champion by champion win-rates, a flavor of the month list, and more. Keep on top of all the latest League statistics, and become a better player!

https://play.google.com/store/apps/details?id=manuele.bryan.lolwinrate
